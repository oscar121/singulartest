# Oscar Perdomo test

This is a Playwright test suit that includes the checking of the Login page, Product page and Checkout process.

# Prequisites
Nodejs
Playwright
Npm

## Installation

### Download from git
```bash
cd "move to your desired folder"
git clone https://github.com/oscar121/singulartest.git
```
Add folder to your text editor of choice.

### Install dependencies with npm
```bash
npm install 
npx playwright test
```

## Run the tests

```bash
npx playwright test
```
## Run the tests wit ui

```bash
npx playwright test --ui
```

## Show report

```bash
npx playwright show-report
```
